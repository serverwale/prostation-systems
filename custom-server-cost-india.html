What drives the cost

The GPU is usually the biggest line item for AI and rendering builds, so size it to your workload first, then balance CPU, RAM and storage around it. ECC memory and redundant power add reliability for 24×7 duty. New builds include full warranty; refurbished can stretch a tight budget for non-critical roles — see new vs refurbished servers.
